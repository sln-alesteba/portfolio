SPLINE ITERPOLATION¶

The following cells contains the linear interpolation models for both regular polynomial and spline transformations. Then visualize the model fit to the points with a varying range of degrees. The higher the degree, both the polynomial and spline, the more accurate the model to the points. In a data science approach a lower degree may be better to avoid overfitting, but if we want to use the model for drawing and graphics purpose, a high degree that fits to the points is a great choice. Enjoy the samples below.

from sklearn.linear_model import Ridge
from sklearn.pipeline import make_pipeline
from sklearn.preprocessing import PolynomialFeatures

# try different degrees to fit polynomials.
for degree in [1,2,3, 10]:
    model = make_pipeline(PolynomialFeatures(degree), Ridge(alpha=1e-3))
    model.fit(x, y)
    y_plot = model.predict(x)
    ax.plot(x, y_plot, label=f"degree {degree}")
